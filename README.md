## How to Host on GitHub Pages

This project is built with static HTML, CSS, and JavaScript, making it perfect for **GitHub Pages**.

### Steps to Host:
1.  **Initialize Git**:
    - Open your terminal in this folder.
    - Run `git init`.
    - Run `git add .`.
    - Run `git commit -m "Initial commit"`.
2.  **Push to GitHub**:
    - Create a new repository on GitHub.
    - Follow the instructions to push your local code to the new repository.
3.  **Enable Pages**:
    - Go to your repository **Settings** on GitHub.
    - Click **Pages** in the sidebar.
    - Under **Branch**, select `main` (or `master`) and click **Save**.

Your store will be live at `https://<your-username>.github.io/<repo-name>/`!
